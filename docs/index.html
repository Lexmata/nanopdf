Quick Start

Add NanoPDF to your project with performance features enabled.

# Cargo.toml
[dependencies]
nanopdf = { version = "0.1", features = ["parallel", "async"] }

# Enable all performance features:
# - parallel: Rayon-based parallelization
# - async: Tokio-based async I/O
# - jpeg2000: JPEG 2000 support
