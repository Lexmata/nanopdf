Run Benchmarks Yourself

# Clone the repository
git clone https://github.com/lexmata/nanopdf
cd nanopdf/nanopdf-rs

# Run all benchmarks
cargo bench

# Run specific benchmark
cargo bench --bench geometry
cargo bench --bench buffer
cargo bench --bench pdf_objects
