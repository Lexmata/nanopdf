API Reference

Core Modules

Document

Main entry point for PDF operations

  • from_bytes(data)
  • page_count()
  • load_page(index)
  • title()
  • close()

Page

  • bounds()
  • render(scale)
  • render_with_matrix(matrix)
  • text()
  • links()

Pixmap

  • height()
  • samples()
  • save_png(path)
  • save_jpeg(path, quality)
  • to_png()

Buffer

Dynamic byte array

  • new()
  • from_bytes(data)
  • len()
  • append(data)
  • clear()
  • slice(start, end)

Stream

Buffered I/O operations

  • open_file(path)
  • open_memory(data)
  • read(count)
  • seek(pos, whence)
  • tell()
  • is_eof()

Colorspace

Color management

  • device_gray()
  • device_rgb()
  • device_cmyk()
  • components()
  • name()

Geometry

Point

2D point (x, y)

  • new(x, y)
  • transform(matrix)
  • distance(other)

Rect

Rectangle (x0, y0, x1, y1)

  • new(x0, y0, x1, y1)
  • width()
  • height()
  • contains(point)
  • intersect(other)
  • union(other)

Matrix

2D transformation matrix

  • identity()
  • scale(sx, sy)
  • rotate(degrees)
  • translate(tx, ty)
  • concat(other)

Quad

Quadrilateral (4 points)

  • from_rect(rect)
  • bounding_rect()
  • transform(matrix)
